Orígenes CORS Permitidos

Define qué dominios de Odoo pueden enviar trabajos de impresión a este servicio. Si la lista está vacía, solo se permite localhost (comportamiento seguro por defecto).

Para permitir cualquier origen (no recomendado), agrega exactamente * como origen.

¿Qué es CORS?
Cross-Origin Resource Sharing es un mecanismo de seguridad del navegador que restringe cómo una página web de un dominio puede interactuar con recursos de otro dominio. Configurar esto previene que sitios maliciosos tomen control de tus impresoras.

Límite de Tamaño de Payload

Tamaño máximo que puede tener la imagen base64 enviada por Odoo POS en cada petición de impresión. Un recibo típico pesa menos de 500 KB.
